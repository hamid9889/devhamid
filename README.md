# Developer Portfolio - devHamid

Welcome to my modern developer portfolio! This is a sleek, responsive portfolio website showcasing my projects, skills, and experience.

## 🎨 Features

- **Modern Dark Theme** - Sleek design with cyan and pink accents
- **Fully Responsive** - Works perfectly on desktop, tablet, and mobile devices
- **Smooth Animations** - Interactive scroll animations and transitions
- **Project Showcase** - Display your best work with descriptions and tech stack
- **Skills Section** - Highlight your technical skills organized by category
- **Contact Links** - Easy ways for people to reach out to you
- **Performance Optimized** - Fast loading and smooth interactions

## 🚀 Quick Start

### GitHub Pages (Recommended)
1. Go to Settings → Pages
2. Select `main` branch as source
3. Your portfolio will be live at: `https://hamid9889hy.github.io/devHamid-portfolio`
